# 경영데이터과학론심화 퀴즈 앱

Orange3 기반 경영데이터과학론심화 과목을 위한 웹 기반 퀴즈 애플리케이션입니다.

## 바로가기
**[퀴즈 시작하기](https://hellworlding.github.io/dsclassquiz/)**

---

## 주요 기능

### 퀴즈
- **범위별 선택**: 6-2, 7-1, 9-1, 9-2, 10-1, 10-2, 11-1
- **복수 범위 선택** 가능
- **문제 순서 섞기** 옵션
- **객관식만 풀기** 옵션

### 문제 유형
| 유형 | 설명 |
|------|------|
| 객관식 | 4지선다 문제 |
| 주관식 | 단답형 (복수 정답 지원) |
| 서술형 | 해설 확인 문제 |

### 난이도 표시
각 문제에는 난이도가 별(★)로 표시됩니다.

| 난이도 | 표시 | 기준 |
|--------|------|------|
| 쉬움 | ★ (녹색) | 단순 암기, 정의 확인, 직관적 판단 |
| 보통 | ★★ (주황) | 개념 이해 필요, 2개 이상 개념 연결 |
| 어려움 | ★★★ (빨강) | 깊은 이해, 응용/비교/분석 요구 |

### 오답노트
- 틀린 문제 **자동 저장**
- **중요 표시** 기능 (정답을 맞춰도 저장 가능)
- 범위별 필터링 / 정렬
- **오답만 다시 풀기** (맞추면 자동 제거)
- **오답 내역 다운로드** (텍스트 파일)

### 편의 기능
- **다크모드** 지원
- **모바일 최적화**
- 데이터 내보내기/가져오기
- 문제 네비게이션

## 사용 방법

1. **범위 선택**: 원하는 범위를 체크
2. **옵션 설정**: 순서 섞기, 객관식만 풀기 선택
3. **퀴즈 시작**: 문제 풀이 시작
4. **오답노트 확인**: 틀린 문제 복습

### 중요 표시 기능
문제를 풀고 나서 **"중요 표시"** 버튼을 클릭하면:
- 정답을 맞춘 문제도 오답노트에 저장됩니다
- 나중에 다시 복습할 수 있습니다

---

## 데이터 저장

모든 데이터는 **브라우저 localStorage**에 저장됩니다.
- 오답노트
- 다크모드 설정

**주의**: 브라우저 데이터를 삭제하면 오답노트도 삭제됩니다.
→ **설정 > 데이터 내보내기**로 백업하세요!
