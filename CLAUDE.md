# 경영데이터과학론심화 퀴즈 앱

## 프로젝트 개요
Orange3 기반 경영데이터과학론심화 과목을 위한 웹 기반 퀴즈 애플리케이션입니다.

## 기술 스택
- **Frontend**: Vanilla JavaScript (ES6+), HTML5, CSS3
- **저장소**: localStorage (오답노트, 설정)
- **배포**: GitHub Pages

## 주요 기능

### 1. 퀴즈 기능
- 범위별 퀴즈 선택 (6-2, 7-1, 9-1, 9-2, 10-1, 10-2, 11-1)
- 복수 범위 선택 가능
- 문제 순서 섞기 옵션
- **객관식만 풀기** 옵션 (주관식/서술형 제외)

### 2. 문제 유형
- **객관식 (mcq)**: 4지선다 문제
- **주관식 (short)**: 단답형 문제 (복수 정답 지원)
- **서술형 (essay)**: 해설 확인 문제

### 3. 오답노트
- 틀린 문제 자동 저장
- **중요 표시** 기능 (정답을 맞춰도 오답노트에 추가 가능)
- 범위별 필터링
- 최근 틀린 순 / 많이 틀린 순 정렬
- 오답만 다시 풀기 (맞추면 오답노트에서 자동 제거)
- 오답 내역 다운로드 (텍스트 파일)

### 4. UI/UX
- 다크모드 지원
- 모바일 최적화 (터치 친화적 44px 버튼)
- Safe Area 지원 (노치 디스플레이)
- 문제 네비게이션 점 표시
- 애니메이션 효과 (정답/오답 피드백)

### 5. 데이터 관리
- 데이터 내보내기 (JSON)
- 데이터 가져오기
- 모든 데이터 초기화

## JSON 데이터 구조
```json
{
  "meta": {
    "range": "6-2",
    "title": "6강 2부",
    "topic": "주제명"
  },
  "questions": [
    {
      "qid": "6-2-Q1",
      "type": "mcq",        // mcq, short, essay
      "prompt": "문제 내용",
      "choices": [          // mcq인 경우만
        {"cid": "A", "text": "선택지 A"},
        {"cid": "B", "text": "선택지 B"},
        {"cid": "C", "text": "선택지 C"},
        {"cid": "D", "text": "선택지 D"}
      ]
    }
  ],
  "answers": [
    {"qid": "6-2-Q1", "correct": "A"}    // 또는 ["답1", "답2"] (복수 정답)
  ],
  "explanations": [
    {"qid": "6-2-Q1", "explanation": "해설 내용"}
  ]
}
```

## 로컬 테스트
CORS 정책으로 인해 `file://` 프로토콜에서는 fetch가 작동하지 않습니다.

```bash
# Python 서버 사용
cd web_version
python -m http.server 8000
# http://localhost:8000 접속

# 또는 VS Code Live Server 확장 사용
```

## 배포 (GitHub Pages)
1. GitHub 저장소 Settings > Pages
2. Source: `main` 브랜치, `/ (root)` 폴더 선택
3. Save 클릭
4. URL: `https://[username].github.io/[repo-name]/`

## 개발 프롬프트 요약

### 사용자 요청 흐름

1. **초기 요청**: "경영데이터과학론심화 퀴즈 앱을 만들어줘. GitHub Pages에 배포할 수 있도록 HTML/CSS/JS로."

2. **오답노트 기능**: "오답노트 기능을 추가해줘. localStorage에 저장하고, 필터/정렬, 오답만 다시 풀기 기능."

3. **Phase 3 UX**: "애니메이션, 다크모드, 데이터 내보내기/가져오기, 모바일 최적화, 그만두기 버튼, 문제 네비게이션 점, 결과 리뷰 기능을 추가해줘."

4. **오답 다운로드**: "오답 내역 다운 버튼을 추가해줘. 시간도 한국 서울 시간을 기준으로."

5. **객관식 필터**: "주관식이랑 객관식이랑 분리하도록 문제 json에 페어를 추가해줘. 그리고 웹페이지에서 객관식만 선택할 수도 있게 해줘."

6. **중요 표시**: "오답 외에도 중요하다고 체크하면 오답노트에 추가되도록 하는 기능을 추가해줘."

### 기술적 구현 사항
- Vanilla JavaScript 클래스 기반 아키텍처 (QuizApp)
- CSS Custom Properties를 활용한 다크모드
- localStorage API로 영속적 데이터 저장
- Fetch API로 JSON 데이터 로드
- toLocaleString()으로 한국 시간대 포맷팅
- @keyframes 애니메이션
- CSS safe-area-inset으로 노치 대응
- 터치 디바이스 최적화
